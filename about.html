Project Overview

This website serves as the Milestone #1 deliverable for IT 3203. The project aims to structure and present detailed research on the evolution of HTTP, from its rudimentary start to the latest QUIC implementation, using fundamental web technologies.

Project Goal:

  1. Establish a clean, professional, and consistent multi-page structure.
  2. Utilize core HTML features (headings, lists, tables).
  3. Implement custom CSS for a business-style aesthetic.
  4. Prepare a solid foundation for adding JavaScript interactivity (Milestone #2) and full responsiveness (Milestone #3).
